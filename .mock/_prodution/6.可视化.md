# 可视化渲染

技术选型：

- 图形数量少，且样式简单 dom + css 直接实现
  实现简单，加载快

- 图形数量多，svg
  简单直观方便

- 图形较为复杂，canvas2d
  绘制性能好

- 数量很多，达到万级别，webgl
  支持批渲染，性能很高

## svg

<defs> 元素用于定义可重用的图形内容。
这些内容在定义时不会直接显示在图形中，但可以通过引用 <defs> 中的元素来在 SVG 文档的其他部分使用它们
